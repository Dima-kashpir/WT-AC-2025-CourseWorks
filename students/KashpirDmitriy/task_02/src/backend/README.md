## Требования

- Python 3.11

## Как запускать

### 1. Создание виртуального окружения

В корне проекта папки Kashpir прописать:

```Bash
python3.11 -m venv venv
```

> Здесь `venv` — имя папки с виртуальным окружением.
> Её можете назвать иначе, но все дальнейшие команды предполагают именно это имя.

### 2. Активация виртуального окружения

macOS / Linux:

```bash
source venv/bin/activate
```

Windows (PowerShell):

```bash
venv\Scripts\Activate
```

### 3. Установка зависимостей

После активации окружения выполните:

```bash
pip install -r requirements.txt
```
